# 濮阳同城视频交友平台（纯前端静态版）
濮阳本地青年视频相亲、线上连麦交友、短视频分享网站，模拟直播房间、实时视频通话、短视频滑动功能，使用localStorage缓存用户、弹幕、点赞数据，无需后端。
## 功能清单
1. 毛玻璃导航、渐变直播风视觉，手机/电脑完美适配
2. 在线视频直播间：弹幕滚动、虚拟礼物、在线人数、静音/挂断模拟
3. 短视频瀑布流，上下滑切换、点赞收藏评论
4. 濮阳区县筛选同城用户，一键发起视频邀约
5. 主播个人主页、开播入口、资料编辑
6. 线下视频相亲专场活动展示
## 运行方式
1. 本地打开index.html直接预览
2. 上传GitHub开启Pages免费线上访问
